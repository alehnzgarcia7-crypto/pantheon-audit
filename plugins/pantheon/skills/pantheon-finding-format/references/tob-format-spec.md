# PANTHEON Finding Format (TOB-extended)

### PANTHEON-NNNN: <Short Title (max 80 chars)>

**Metadata**: Audit ID, Discovered by, Phase, Discovered at, Status.
**Severity**: Severity, CVSS v4.0 vector + score, Exploitation Difficulty, Business Impact USD, Threat Actors Capable [T1-T8].
**Affected**: Component, File(s) path:LINE-LINE, Function/Endpoint, First introduced commit.
**Description**: 2-4 paragraphs technical.
**Exploitation Scenario**: Numbered steps with code, curl, request/response.
**Recommendation Short-term**: Specific change with diff.
**Recommendation Long-term**: Architectural improvement.
**Compliance Mappings**: JSON with all 18 framework fields.
**Adversarial Gate**: Reviewed by, Verdict (ACCEPT/REVISE/WITHDRAW/DISPUTE), Confidence %, PoC verified, Cross-verified by.
**References**: CWE, OWASP, CVE, papers, internal docs.
