# STRIDE per Element

Microsoft. Per DFD element apply STRIDE.

## Categories
- Spoofing identity
- Tampering with data
- Repudiation
- Information disclosure
- Denial of service
- Elevation of privilege

## Element type applicability
- External entities: S, R
- Processes: all six
- Data stores: T, R, I, D
- Data flows: T, I, D
